# Breathline – Centralized Medical Records Platform

A full-stack web application for secure medical record management 
with role-based dashboards for patients, doctors, and emergencies.

## Live Demo
https://breathline-project-showcase.vercel.app/

## Features
- Patient dashboard to manage and share medical records
- Doctor dashboard with licensed access to shared patient records
- Emergency dashboard for instant medical data access
- Secure role-based authentication and access control

## Tech Stack
Next.js, MongoDB, Cloudinary, Tailwind CSS, JavaScript

## How to Run
1. Clone the repo
2. Install dependencies: npm install
3. Add your .env variables (MongoDB URI, Cloudinary keys)
4. Run: npm run dev
